## About

iSearch is a side project I took up to keep my React skills fresh. It allows users to search iTunes for their favorite shows, movies, musicians, actors, etc. (Anything that the iTunes API provides). 

## Tech Stack

* JavaScript
* React
* Jest/Enzyme
* SASS
* Bootstrap
